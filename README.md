
# AXI-TECH / Gestion Destock

Ce projet permet de gérer le déstockage des produits de manière optimisée.

## Fonctionnalités
* Suivi des stocks en temps réel
* Alertes en cas de niveau critique
* Historique complet des mouvements de marchandises

## Installation
Pour installer et lancer le projet en local, exécutez la commande suivante :
\`\`\`bash
npm install && npm start
\`\`\`
